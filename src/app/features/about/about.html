I'm a Full Stack Developer with 3+ years of experience building enterprise-grade financial and business applications in the FinTech domain — from customer onboarding and cross-border remittance to operational admin and reporting systems.

I deliver end-to-end solutions across Angular, React, Next.js, and Node.js, with strong hands-on work in secure eKYC onboarding, RBAC admin panels, transaction & remittance platforms, and real-time dashboards. On the backend, I design REST APIs with JWT authentication, middleware, and PostgreSQL / MongoDB — and build real-time features with Socket.io and WebSockets, as shown in my Pulse chat app and Analytics Dashboard live demos.

Beyond production FinTech work, I've built portfolio projects including a multi-tenant Kanban platform (NgRx Entity, CDK DragDrop), a user registration & profile system (Angular + MongoDB), and performance-focused UIs using RxJS, OnPush change detection, lazy loading, and reusable component libraries. I thrive in Agile teams, prioritize clean architecture, and stay current with modern patterns — Angular Signals & Standalone Components, React Hooks & Context, and Next.js App Router, Server Components, and SSR/SSG.
